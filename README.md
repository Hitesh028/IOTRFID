# IOTRFID
Building IOTRFID system with Inventory

# Hardware requirement
ESP8266 Nodemcu <br/>
125KHz RFID Reader RC522 <br/>
Raspberry pi <br/>
Jumper wires <br/>
125KHz RFID Tags <br/>

# Tools required
Arduino IDE <br/>
Openssl <br/>
MQTT Broker <br/>
MySql <br/>
